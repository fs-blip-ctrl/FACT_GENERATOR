# 🌍 Country Facts Generator

## 📌 Description
The Country Facts Generator is a web-based application that displays random information about countries around the world. It fetches real-time data using a REST API and presents it in an interactive and user-friendly format.

---

## 🚀 Features
- 🌍 Displays random country information  
- 🏙 Shows capital of the country  
- 🌐 Displays region  
- 🎲 Generates a new country on each click  
- ⚡ Uses real-time data from API  

---

## 🛠 Technologies Used
- HTML  
- CSS  
- JavaScript  
- REST API
